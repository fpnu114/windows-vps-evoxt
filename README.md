## $5 Windows VPS 能干什么？不能干什么？

先把期望值对齐，省得买回来骂娘。

### ✅ 能干的事

- **跑轻量级挂机脚本**：Python、PowerShell 脚本，只要不是高并发的，$5 方案完全够用
- **量化/外汇交易软件**：MT4/MT5 这类软件对 CPU 要求不高，主要靠稳定的网络和不间断在线，$5 的 VPS 跑起来没啥问题
- **RDP 远程桌面**：日常的网页浏览、文件操作、远程办公，连接稳定之后体验挺好的
- **简单的 Web 服务**：跑个小型网站、API 接口，流量不大的话没问题
- **开发测试环境**：临时测代码、调试环境，便宜好使

### ❌ 不适合的场景

- 高并发 Web 应用（流量一上来就扛不住）
- 大型数据库（内存不够用）
- 视频转码、渲染类任务（CPU 核心数太少）
- 多人同时 RDP 连接（Windows Server 默认只允许 2 个并发连接，多用户需要额外授权）
